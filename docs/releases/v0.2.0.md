# ai-gram v0.2.0

`v0.2.0` expands Bot API coverage after `v0.1.1` with additional send/media methods, media groups, bot commands/menu, invite/join request support, and admin management methods.

## Highlights

- Added more safe send methods:
  - `SendContact`
  - `SendLocation`
  - `SendVenue`
  - `SendPoll`
  - `StopPoll`
  - `SendDice`
- Added media methods:
  - `SendSticker`
  - `SendAnimation`
  - `SendVideoNote`
  - `SendMediaGroup`
- Added bot commands and menu methods.
- Added invite link methods.
- Added chat join request methods and dispatch helpers.
- Added admin management methods.
- Added v0.2 live smoke scripts for safe send/media group flows.
- Public docs remain English; user/operator reports remain Russian.

## Added

### Send methods

- `SendContact` for sending typed contact messages.
- `SendLocation` for sending geographic coordinates and optional live-location fields.
- `SendVenue` for sending venue coordinates, title, address, and provider metadata.
- `SendPoll` and `StopPoll` for poll creation and poll closing.
- `SendDice` for Telegram dice emoji sends.

### Media and media groups

- `SendSticker` with `FileID`, `FileURL`, and `FileUpload` support.
- `SendAnimation` with file refs/uploads, captions, spoilers, thumbnails, reply parameters, and reply markup.
- `SendVideoNote` with file IDs/uploads, thumbnails, duration/length fields, reply parameters, and reply markup.
- `SendMediaGroup` with `InputMediaPhoto`, `InputMediaVideo`, `InputMediaAudio`, and `InputMediaDocument`.
- Multipart `attach://` handling for media group uploads and thumbnails with deterministic attachment names.

### Bot commands/menu

- `SetMyCommands`, `DeleteMyCommands`, and `GetMyCommands`.
- Typed command scopes for default, private chats, group chats, administrators, chats, chat administrators, and chat members.
- `SetChatMenuButton` and `GetChatMenuButton` with typed commands/default/Web App menu buttons.
- `SetMyDefaultAdministratorRights` with typed administrator rights.

### Invite links and join requests

- `ExportChatInviteLink`, `CreateChatInviteLink`, `EditChatInviteLink`, and `RevokeChatInviteLink`.
- `ApproveChatJoinRequest` and `DeclineChatJoinRequest`.
- Minimal `telegram.ChatInviteLink` and `telegram.ChatJoinRequest` types.
- `chat_join_request` update decoding and dispatcher helpers.

### Admin management

- `PromoteChatMember`.
- `SetChatAdministratorCustomTitle`.
- `SetChatPermissions`.

These methods are admin/state-changing operations and are intentionally documented as manual-only for live verification.

### Live smoke and testing docs

- `scripts/smoke_v02_send_methods.sh` and `examples/v02_send_methods` for contact/location/venue/poll/dice live smoke.
- `scripts/smoke_media_group.sh` and `examples/media_group_smoke` for `SendMediaGroup` live smoke with generated upload fallback.
- Updated API coverage, roadmap, v0.2 checkpoint, manual testing, and live smoke matrix docs.

## Safety

- Admin/state-changing methods are unit/httptest covered and documented as manual-only.
- Destructive/admin live checks were intentionally not auto-run.
- Secrets and token-bearing URLs are not printed by the smoke scripts or reports.
- The raw `Bot.Token()` public accessor remains absent.
- `.env.local` and `.deploy/generated.env` remain ignored.

## Known limitations

- `ai-gram` does not provide full Telegram Bot API coverage yet.
- Optional sticker, animation, and video note live media checks are skipped unless matching media environment variables are provided.
- Admin/state-changing methods require dedicated test chats and explicit confirmation before live testing.
- Remaining future areas may include:
  - chat management methods
  - forum topics
  - reactions
  - inline mode
  - sticker set management
  - payments, Passport, Games, Business APIs, Stars/gifts, and other advanced areas

## Installation

Use:

```bash
go get github.com/xDilettante/ai-gram@v0.2.0
```

The `v0.2.0` tag is published, and this command has been verified from an external temporary project.
